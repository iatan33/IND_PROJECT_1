# MedEmoc — Веб-застосунок для медичної клініки

Індивідуальний проєкт №1 з дисципліни «WEB-орієнтовані технології. Frontend розробка».

## Опис проєкту
Цей репозиторій містить Frontend-частину вебсайту приватної медичної клініки «MedEmoc». Застосунок розроблено для надання користувачам інформації про послуги клініки, лікарів та графік роботи, а також для забезпечення можливості онлайн-запису на прийом.

Веб-застосунок є повністю адаптивним та коректно відображається на десктопних, планшетних та мобільних пристроях.

## Інструкція із запуску
